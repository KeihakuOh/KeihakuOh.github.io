Full Stack Engineer at CUUSOO SYSTEM (Tokyo, JP) — May 2023 - Present

As a part-time Full Stack Engineer, I have been involved in a project aimed at increasing user engagement and expanding the user base. I have utilized technologies such as Ruby (Ruby on Rails), JavaScript (React, Vue.js), Python, Docker, and MySQL to develop new features and enhance existing functionalities. Below are some key highlights of my contributions:

  • Implementing Two-Factor Authentication: To enhance user trust and security, I implemented a two-factor authentication (2FA) system using Ruby and Ruby on Rails. The authentication code was generated using a cryptographically secure random number generation algorithm, SecureRandom, ensuring robust security standards.
  • Developing Reply-to-Reply Feature for Comments: To support our user growth strategy, I worked on adding a reply-to-reply feature to the existing commenting system. Given the high load on the current server, we optimized the database by distinguishing between comment replies and replies to replies using a flag within the comments table, rather than creating new tables, thus maintaining database efficiency.
  • Implementation of a Comment Notification Settings System: Initially, I implemented a feature to notify users whenever a related article was commented on. However, it was determined that not all users needed such notifications every time. To address this, I improved the functionality to allow users to configure whether they want to receive notifications. This implementation was done using Ruby and Ruby on Rails. Additionally, since there were many common settings, I created a parent class and implemented inheritance from it. Furthermore, I considered different test cases depending on the device and conducted testing accordingly.
  • Implementation of a System to Display Company Information on a Map: To provide a clear visual representation of where companies are located within a specific area, I implemented a system that displays company information on a map. Additionally, I developed a user interface for inputting company location data and the backend logic to send and save that data. The system was built using Vue.js and Ruby on Rails.

Infrastructure Engineer at Waseda University (Tokyo, JP) — April 2023 - Present

While in my fourth year as an undergraduate student and studying for the Applied Information Technology Engineer Examination, I developed a strong interest in networks and security. To gain practical experience, I joined the lab's server administration team, and in the spring of my first year as a Master's student, I took on the role of leader. My primary responsibilities include assembling and maintaining GPUs for research purposes and building and managing the laboratory network infrastructure. Key achievements include:

  • Upgrading Security Protocols and Rebuilding Servers: Recognizing the need for enhanced security, I upgraded the lab's server communication from HTTP to HTTPS by issuing a certificate myself. I also rebuilt the outdated Prometheus monitoring server from scratch and utilized Kubernetes for easy HTTPS communication setup. Additionally, I rebuilt the lab's outdated pfSense VPN server from scratch, implementing redundancy for improved reliability.
  • Establishing Secure Guest Wi-Fi and Network Segmentation: Noticing that visitors could only access the university's Wi-Fi, which led to slow communication, I set up a dedicated guest Wi-Fi for the lab. I also configured a firewall to ensure that guests do not have access to the internal network, thus maintaining network security.
  • Solving Storage Limitations and Implementing Log Retention Solutions: To address the recurring issue of "insufficient space for research environments," I mounted an SSD to expand storage capacity. Furthermore, recognizing the importance of log retention for research purposes, I am currently in the process of setting up an MLflow server to manage and analyze logs efficiently.
